# Source mapping

This supplement reorganizes the uploaded experiment code into stable, reviewer-facing entrypoints.

| Supplement file | Role |
|---|---|
| `src/retained_subspace/glue_experiment.py` | GLUE/text classification experiments, including MNLI, QNLI, MRPC, SST-2, CoLA, QQP, main low-budget results, oracle diagnostics, and appendix audits. |
| `src/retained_subspace/vision_experiment.py` | Vision/image-classification experiments, including CIFAR-10, CIFAR-100, Food-101, EuroSAT, main low-budget results, oracle diagnostics, and appendix audits. |
| `scripts/run_main_text_and_vision.sh` | Default compact main-text GLUE/CIFAR run wrapper. |
| `scripts/run_additional_benchmarks.sh` | Optional wrapper for SST-2, CoLA, QQP, Food-101, and EuroSAT. |
| `scripts/run_oracle_diagnostics.sh` | Exact small-budget subset-oracle diagnostic wrapper. |
| `scripts/run_appendix_audits.sh` | Split-seed and appendix audit wrapper. |
| `llm_sanity/gsm8k_qwen_math7b_experiment.py` | Qwen2.5-Math-7B GSM8K retained-subspace sanity experiment. |
| `llm_sanity/gsm8k_mistral_larger_pool_experiment.py` | Mistral-7B GSM8K larger-pool sanity checks at larger LoRA ranks. |
| `llm_sanity/humanevalplus_qwen_coder7b_experiment.py` | Qwen2.5-Coder-7B HumanEval+ retained-subspace experiment. The paper-facing metric is EvalPlus plus-suite pass@1. |
| `llm_sanity/humanevalplus_qwen_coder7b_launcher.py` | Fixed-argument launcher for the HumanEval+ pass@1 experiment. |
| `tools/collect_results.py` | Recursive result collector for CSV summaries generated by the experiment scripts, including HumanEval+ pass@1 summaries. |

The original temporary filenames were not kept in the main package structure. Method names such as `v5` remain inside command-line options because they are used by the current experiment code to refer to the URS score variant.
